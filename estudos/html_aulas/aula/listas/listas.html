Listas

Vamos usar agora teag's sobre listas ordenadas e não ordenadas

Listas ordenadas usamos a tag <ol>

Temos marcadores que podemos definir em seu parâmetro TYPE tais como:

  1. 1 - CRIA LISTAS NUMERADAS
  2. A - CRIA LISTAS COM LETRAS MAIÚSCULAS
  3. a - CRIA LISTAS COM KETRAS MINÚSCULAS
  4. I - CRIA LISTAS COM Nº ROMANOS MAIÚSCULOS
  5. i - CRIA LISTAS COM Nº ROMANOS MINÚSCULAS

Listas náo ordenadas usamos a tag <ul>

Temos marcadores que podemos definir em seu parâmetro TYPE tais como:

  • disc - UMA BOLA COMPLETAMENTE PINTADA
  • circle UMA BOLA SEM PREENCHIMENTO
  • square UM QUADRADO PREENCHIDO

Internamente usamos a tag <li> que pode ter ou não o seu fechamento, ele não é necessário em HTML5

Listas de definição

Podemos usar listas para poder definir termos com a tag <dl>

HTML
Línguagem de marcação usado para construção de conteúdo do site
CSS
Linguagem de marcação para a criação de estilos de um site
JasvaScript
Linguagem de programação usada para a criação de interatividade dentro do site.
